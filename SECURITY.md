# Security Policy

## Supported Versions

Only the latest version of RankerHub is actively supported and maintained with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of RankerHub seriously. If you find a vulnerability, please do not open a public issue. Doing so exposes the project and its users to potential risks before a patch can be developed.

Instead, please report security vulnerabilities by following these steps:

1. **Email the Maintainers**: Send a detailed email to [indresh.sharma404@gmail.com].
2. **Include Details**:
   - A clear description of the vulnerability.
   - The steps required to reproduce the issue.
   - Any potential impact or exploits.
   - (Optional) A proposed fix or proof of concept.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to coordinate a security release to resolve the issue as quickly as possible. Thank you for helping keep RankerHub secure!
